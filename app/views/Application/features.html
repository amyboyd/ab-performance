Features

  • Tests are written in CSS. All web developers already know how to write CSS, so there is no learning curve. Here is a example test: 

    ${''.colorHTML()}
  • Debugging is easy - just run abperf.debug() in the Chrome or Firebug console. You can run it right now.
  • The JavaScript file is less than 15 KB. That is about the same as Google Analytics. This small file will not slow down your pages. Also, when parts of our JavaScript are no longer needed, they are deleted from the heap, reducing memory usage in your users' browsers.
  • The report is delivered within an hour of your page view limit being reached.
  • Supports these browsers: Chrome 5+, Firefox 4+ and Safari 4+. Other browsers are not supported and do not count towards your page view quota.
