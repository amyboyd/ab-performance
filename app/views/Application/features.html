Features

  • Tests are written in CSS. All web developers already know how to write CSS, so there is no learning curve.
  • Debugging is easy - just run abperf.debug() in the Chrome or Firebug console.
  • The JavaScript file is less than 15 KB. That is about the same as Google Analytics. This small file will not slow down your pages.
  • The report is delivered within an hour of your page view limit being reached.
